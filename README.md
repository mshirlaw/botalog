# botalist

A public index of Facebook Messenger bots.

```json
{
  "name": "The Mad Botter",
  "link": "A link to the relevant page",

  "author": "The Mad Hatter",

  "description": "A mad bot for ordering the hippy hoppy hippiest of hats.",
  "rating": 7,
  "categories": [
    "on_demand",
    "productivity",
    "fun"
  ],
  "tags": [
    "hats",
    "hip",
    "i love hats"
  ]
}
```
